# CODSOFT Machine Learning Internship

This repository contains the Machine Learning projects completed as part of the CODSOFT Machine Learning Internship program.

## Tasks Completed

### 1. Spam SMS Detection
- Developed a machine learning model to classify SMS messages as Spam or Ham (Not Spam).
- Used TF-IDF Vectorization for text feature extraction.
- Implemented classification algorithms to detect spam messages.

### 2. Credit Card Fraud Detection
- Built a machine learning model to identify fraudulent credit card transactions.
- Performed data preprocessing and model training using classification algorithms.
- Evaluated the model using performance metrics such as accuracy, precision, and recall.

### 3. Customer Churn Prediction
- Developed a predictive model to identify customers likely to discontinue a subscription-based service.
- Used customer demographic and behavioral data for prediction.
- Applied machine learning techniques to improve customer retention analysis.

## Technologies Used
- Python
- Pandas

- Scikit-learn
- Google Colab
- Machine Learning Algorithms
